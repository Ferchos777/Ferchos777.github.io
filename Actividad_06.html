Siglas

LAN (Local Area Network)

Red de área local que conecta dispositivos en un área geográfica limitada, como una casa, oficina o edificio.

HTTP (HyperText Transfer Protocol)

Protocolo de transporte de archivos en formato HTML (páginas web).

HTML (HyperText Markup Language)

Lenguaje de marcado utilizado para crear y estructurar contenido en la web.

DHCP (Dynamic Host Configuration Protocol)

Protocolo que asigna automáticamente direcciones IP y otros parámetros de red a dispositivos en una red.

SMTP (Simple Mail Transfer Protocol)

Protocolo para la transferencia de correo electrónico en formato simple.

FTP (File Transfer Protocol)

Protocolo utilizado para la transferencia de archivos entre un cliente y un servidor en una red.

POP (Post Office Protocol)

Servicio para acceder a la oficina postal virtual de correo electrónico.

IP (Internet Protocol)

Representa la dirección lógica de un dispositivo; está formada por 4 números en un rango de 0 a 255.

TCP (Transmission Control Protocol)

Protocolo de comunicación que garantiza la entrega ordenada y confiable de datos entre dispositivos.

UDP (User Datagram Protocol)

Protocolo de comunicación que permite el envío rápido de datos sin garantizar su entrega o orden.
